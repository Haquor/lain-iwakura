# lain-iwakura

### Installation

lain-iwakura requires [Node.js](https://nodejs.org/) to run.

Create config.js with your Discord bot's token
```
{
  "token"  : "e0ae78ceb038fae5344ccacd869a3fec",
  "prefix" : "+"
}
```

Install the dependencies and start the bot.

```sh
$ cd lain-iwakura
$ npm install -d
$ node lain.js
```
